6.1 Setting

This section is currently being revised as I read and process the recent DiD and event study literature. Some statements may be unclear or incorrect, which I will try to refine them as I go.

We can conceptualise the empirical setting as an event study, where treatment occurs when a location experiences a sewage spill above a defined threshold, and house prices serve as the outcome variable.

  • Absorbing vs. non-absorbing treatment: Treatment status can be defined in two ways. In a non-absorbing framework, locations may alternate between treated and control states, depending on spill severity in each period. For example, a location may experience a large spill in one quarter and no spill in the next quarter. Alternatively, in an absorbing framework, a location is considered treated from the first period it exceeds the threshold and remains treated thereafter.
  • Treatment is staggered: Spill events, and therefore treatment, occur at different times across different locations.
  • Treatment measurement: Spill severity is measured by either the number of incidents or by total duration in a given period \(t\) (e.g., monthly or quarterly). While these measures are continuous, we can construct a binary treatment indicator by classifying a house as treated if it lies within radius X of a spill site whose severity exceeds a predetermined threshold.
  • Fixed effects: To (partially) address confounding, we include both time and location fixed effects. This ensures that identification relies on within-location variation over time, controlling for any time-invariant differences across locations and common shocks across periods.

Given the potential problems with the vanilla TWFE presented in the recent literature, I decide to use the Chaisemartin & d’Haultefoeuille (2024) estimator using the package DIDmultiplegtDYN. This estimator allows for a wide range of settings, including non-absorbing treatment, staggered treatment, continuous treatment, and many other settings that I still need to understand.

6.2 Site-Level Event Study Regressions Within Spill Buffers

The following figures present the dynamic effects of sewage spills on house prices within an X-metre radius around spill sites. For each spill site and quarter, I aggregate spill statistics and average house prices within the corresponding X-metre buffer.1 The estimated regression coefficients capture the within-buffer change in average house prices over time, relative to untreated periods, conditional on the buffer experiencing a high-severity spill in a given quarter. That is, they reflect how prices evolve in a given spill site’s buffer following the occurrence of a high-severity spill within that buffer.

The regressions below are estimated using the Chaisemartin & d’Haultefoeuille (2024) estimator using the package DIDmultiplegtDYN. This estimator allows for a wide range of settings, including non-absorbing treatment, staggered treatment, continuous treatment, and many other settings that I still need to understand. In the event study regressions, the effects are estimated for:

  • Same switchers only. This ensures that event-study effects are estimated only using switchers for whom all the event-study effects can be estimated. This avoids the problem that if treatment effects are heterogeneous across spill sites where treatment switches, changes in the estimated effects across different time horizons might reflect changes in the composition of the sample rather than actual dynamic treatment effects.
  • Normalised. This scales the estimates to represent effects per unit of treatment rather than total effects. For spill sites, this means showing the impact per unit of treatment intensity, making estimates comparable across time periods and sites.

6.3 House-Level Event Study Regressions Within Admin Boundaries

The advantages of using house-level data relative to site-level data in the event study regressions are:

  • House-level controls: Working at the house-level allows us to include property attributes (e.g. newly built, property type etc.) as controls. This ensures that any post-spill changes in the composition of sales do not confound price effects.
  • No double-counting from overlapping buffers: Each property appears only once, even if it falls within 500 m of multiple outfalls. This avoids mechanical correlation among treated observations, which could inflate standard errors.
  • Larger control group: Properties outside of the X-metre buffer remain in the sample. We can therefore use different location fixed effects (e.g., LSOA, MSOA, Admin Ward, Admin County, County, Region, Country, NHS HA) to account for any pre-existing differences in house prices across the study area.

The figures below show the dynamic effects of sewage spills on house prices across different administrative boundaries. Using geocoded transaction data, I calculate the total volume of sewage spills within an Xm radius of each property for every calendar quarter. A property is classified as treated in a given quarter if the spill volume exceeds a specified threshold. This approach allows treatment timing to vary across properties, depending on the occurrence and intensity of nearby spills.

The regressions estimate changes in house prices over time within the same location, comparing treated and untreated periods. Regressions are estimated both without controls (NC) and with controls (C):

  • Old/New: A binary variable indicating whether the property was a newly built property that had not been previosly sold.
  • Property Type: A categorical variable indicating the type of property (detached, semi-detached, terraced, flats/maisonettes, other)
  • Duration: A binary variable indicating the tenure of the property (freehold, leasehold)

Regressions are estimated using the event-study estimator developed by Sun and Abraham (2021).
